# sp Hybridization in Ethyne — Interactive Chemistry EPUB 3 Demonstration

An interactive EPUB 3 learning module on sp hybridization and bonding in ethyne
(C2H2), combining explanatory chapters, MP4 video, embedded WebGL/Babylon.js 3D
viewers (GLB models) and an 8-question self-check quiz. Designed to be read in the
companion **Edu Ebook Reader** (a DRM-free PWA that permits scripted and 3D EPUB
content); general-purpose EPUB readers often restrict scripting and WebGL.

- All chemistry content, 3D models and quiz questions are original work by the author.
- The 3D models were created in Autodesk 3ds Max and exported to GLB for the web.
- The EPUB is intentionally DRM-free for educational use.
