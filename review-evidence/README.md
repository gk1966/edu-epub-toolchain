# Review evidence

This directory contains the three raster figures used as implementation evidence in the review manuscript:

- `figure2-reader.png`: the implemented Reader displaying an embedded interactive 3D chapter;
- `figure3-creator.png`: the implemented Creator authoring workspace;
- `figure5-test-output.png`: a report generated from the bundled `static-smoke-test.js` and `preview-security-test.js` scripts.

The test report records implementation regression checks only. It is not a user study, an accessibility or security certification, an EPUBCheck result, or evidence of learning effectiveness.
