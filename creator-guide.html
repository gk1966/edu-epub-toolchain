Six steps

From blank project to a valid EPUB 3.

1 · Open the Creator

It runs locally in your browser. Tip: serve it over HTTP(S) for the smoothest experience.

2 · Set the book details

Enter title, author, license and choose the book language (about 50 languages). Optionally set a cover image.

3 · Add & structure chapters

Add chapters in the left panel. Write in the Visual editor or in XHTML source, with a live Preview. Use the searchable block library (callouts, definitions, quizzes, tables, layouts, buttons, …).

4 · Insert media & interactivity

Add images, audio and video as managed assets, and embed interactive HTML or 3D/WebGL chapters.

5 · Import existing material

Use the Import… menu to bring in a Word (.docx), Markdown or plain‑text file, an existing EPUB (chapters + images), or an HTML package. Images are embedded automatically as assets.

6 · Preview & export

Check the preview, then Export EPUB. The output is a valid EPUB 3 (navigation document, EPUB 2 NCX for backward compatibility, accessibility metadata, organised assets) that opens in the bundled Reader.

Tip: import a Word document with headings, lists and pictures — the Creator keeps the structure and embeds the images, ready to export as a chapter.
