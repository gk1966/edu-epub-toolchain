Edu EPUB Creator Author accessible EPUB 3 learning books

Install the application on your device

Author accessible, interactive educational EPUB 3 books locally, with an offline app shell and support for desktop, tablet and mobile use.

Open without installing

Press the install button. If no prompt appears, use the instructions for your device.

Android / Chrome

  1. Press “Install application”.
  2. In the prompt, press Install.
  3. Open Edu EPUB Creator from the new icon on the home screen.
  4. If no prompt appears, open the Chrome menu and choose Add to Home screen or Install app.

iPhone / iPad

  1. Open the page in Safari.
  2. Tap the Share button.
  3. Choose Add to Home Screen.
  4. Tap Add to place the icon on the home screen.

Desktop

  1. Open the page with Chrome or Edge.
  2. Press “Install application”.
  3. If no prompt appears, use the install icon in the address bar or the browser menu.
  4. After installation, open the app from the Start Menu, Dock or desktop.
