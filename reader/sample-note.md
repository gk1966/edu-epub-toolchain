# Demo Markdown μαθήματος

Αυτό είναι ένα μικρό αρχείο Markdown για δοκιμή του multi-format reader.

## Στόχοι

- Άνοιγμα σε κινητό, tablet και υπολογιστή.
- Ασφαλής προβολή τοπικού κειμένου.
- Βασική μορφοποίηση χωρίς εκτέλεση HTML.

**Ερώτηση:** ποιο format είναι καλύτερο για διαδραστικότητα;

Η απάντηση είναι `EPUB 3`.

[Εξωτερικός σύνδεσμος](https://example.com) εμφανίζεται ως μπλοκαρισμένο link.
